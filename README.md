# Quizlet Embed Generator

A lightweight, single-file HTML utility that lets you study Quizlet sets in **Learn**, **Match**, **Flashcards**, or **Test** modes entirely through an embedded iframe. Embedded iframes, bypass the default limitations/paywalls on standard study modes allowing you to use any mode indefinitely.

## Features

* **Bypass Study Limits:** Embedding the Quizlet frame allows infinite use of the "Learn" mode.
* **Multiple Modes Supported:** Toggle easily between Learn, Match, Flashcards, and Test.
* **True Full Screen:** Includes a dedicated fullscreen button that stretches the study layout to fill your entire monitor without awkward sizing bugs.
* **Instant Loading:** Just paste any standard Quizlet set URL and press Enter.

## How to Use

1. **Save the Code:** Copy the HTML code into a file named `quizlet-generator.html`.
2. **Open the File:** Double-click the file to open it in any web browser (Chrome, Firefox, Edge, Safari, etc.).
3. **Load a Set:** 
   * Copy the URL of the Quizlet flashcard set you want to study.
   * Paste it into the input box.
   * Choose your preferred study mode from the dropdown menu.
   * Click **Load Study Set** (or hit **Enter**).
